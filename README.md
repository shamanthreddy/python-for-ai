# python-for-ai

This repository contains Python code samples and scripts for Artificial Intelligence (AI) and data analysis projects.

## Features

- 100% Python codebase, focused on practical AI and data workflows
- Example scripts for:
  - API interactions (`apis.py`)
  - Common functions and utilities (`functions.py`)
  - Data retrieval routines (`get_data.py`)
- Organized directories for data resources (`data/`) and case studies or analysis projects (`sales_analysis/`)
- Ready-to-use demos: `hello.py`, `interactive-demo.py`, and more
- Example requirements listed in `requirements.txt`
- Visual outputs, such as `weather_chart.png`

## Project Structure

```
.
├── apis.py
├── functions.py
├── get_data.py
├── data/
├── sales_analysis/
├── requirements.txt
├── [various Python scripts for experiments and analysis]
```

## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/shamanthreddy/python-for-ai.git
   cd python-for-ai
   ```
2. Install required Python packages:
   ```bash
   pip install -r requirements.txt
   ```
3. Run any of the example scripts:
   ```bash
   python hello.py
   ```
